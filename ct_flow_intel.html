Aggressor convention (empirically verified on this feed): SetByAsk prints sit ~0.3–0.7 pts below SetByBid prints in the same minute across 28,800+ minute-windows, and daily net-SetByBid volume correlates +0.69 with settle change. Therefore SetByBid = buyer-initiated (at the offer), SetByAsk = seller-initiated (at the bid). All signing on this page uses that mapping. Leg prints on options are signed by de-drifted IV re-mark; futures leg/EFP/EFS prints are excluded from signed flow.

Dealer gamma inventory map

Bars: open-interest gamma per strike in futures-equivalent lots per 1.00c move (Bloomberg strike gammas × OI). Color = inferred dealer position from cumulative signed customer flow (firm bid/offer prints only) in the blotter since 2026-05-03: customers net bought a strike → dealer short there (short gamma: expect acceleration + IV bid as spot approaches); customers net sold → dealer long (long gamma: expect pinning + IV offered). Volume / open-interest pressure

Latest settle date, per strike: volume vs prior OI, ΔIV, and the repo PVO classifier (New Longs / New Shorts / Closing / Stealth). Event = volume > 50% of prior OI with IV +0.5v — historically these days precede larger moves (magnitude, not direction).
CT.OPT / SENT-09

Net option sentiment (NOSI)

Repo formula: NOSI = Σstrikes, vol≥500 K·ΔIVK·(callVol−putVol) / (F·ΔATM-IV) — plus a signed variant using blotter buy/sell instead of raw volume: Σ K·ΔIVK·netSignedVolK / (F·ΔATM-IV). Signal strength is calibrated to this dataset's own |NOSI| distribution (>80th percentile = Strong); readings with |ΔATM|<0.15v are denominator-unstable and dimmed.
